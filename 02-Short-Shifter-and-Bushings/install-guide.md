# Short Shifter + Bushings Install Guide — Acuity 3-Way + Cable Bushings

## Status: ALREADY INSTALLED — reference and re-install notes

Both parts are on the car. This guide exists so (a) if I pull the shifter to tweak throw or swap bushings I'm not re-learning from scratch, and (b) when I'm eventually in there for a clutch-hydraulics or trans job, the trans-side bushing procedure is already captured. See [overview.md](overview.md) for parts and rationale.

---

## Reality Check

Honestly, one of the easier wins on the whole build. The shifter swaps in under the center console — bolts and a circlip. The trans-side bushings are a bit fussier because you're working by feel from below, but nothing here is intimidating. No fluids, no torque-critical engine fasteners, no harness work.

| Metric | Reality |
|--------|---------|
| Difficulty | Easy-intermediate |
| Time (first attempt) | 2-3 hours |
| Time (re-install or throw adjust only) | 30-45 min |
| Worth DIY? | **Yes.** Shop labor is $150-250; you can do this in an afternoon. |
| Special tools | None exotic — standard hand tools + snap-ring pliers |
| Reversible? | Fully. Keep OEM shifter + bushings bagged. |

The one real bite: the trans-side cable end clips are small, under tension, and you're working by feel. Drop one onto the subframe crossmember and you'll waste 20 minutes with a magnet. Have spares.

## Tools Required

**Critical:**
- 10mm socket + ratchet (center-console fasteners)
- 12mm and 14mm sockets (shifter base + trans-side cable bracket)
- Needle-nose pliers (C-clips on pivot and cable ends)
- Snap-ring / circlip pliers, internal + external (HF $15 kit is fine)
- 1/4" drive torque wrench, 5-25 ft-lb (shifter base bolts — 17 ft-lb per [torque-specs.md](../docs/torque-specs.md))
- Plastic trim pry tool (blue plastic set; metal will mar the console)
- Headlamp (trans-side work is by feel)

**Also useful:**
- Jack + 2 jack stands or ramps (for trans-side access)
- Small parts magnet (for the clips I will drop)
- Zip-lock bags + Sharpie (console has multiple screw lengths — bag each group)
- Silicone grease, NOT lithium (petroleum greases swell poly bushings)

No lift, no transmission jack, no specialty Honda tool.

## Step-by-Step

### Phase A: Center Console Removal (20-30 min)

1. Trans in NEUTRAL, e-brake up, key OUT. Disconnect battery (-) if paranoid about SRS light — this is nowhere near an SRS module, but it's 2 minutes of insurance.
2. Unscrew shift knob counter-clockwise (hand tight from factory).
3. Pry up the shift boot / bezel trim with the plastic tool. Spring clips all around, no screws. Work in from one corner.
4. Pull the ashtray / small storage tray; remove the 2 screws inside.
5. Lift the armrest, remove 2 screws in the cupholder well.
6. Pry up the rear console trim, remove the 2 screws underneath.
7. Slide console back and up. Watch for:
   - 12V outlet wiring (driver side) — unclip at the back, don't pull on the wire
   - Shift boot surround trim — separate piece on some trims
8. Set console aside on a towel. **Bag and label each screw set.**

### Phase B: Shifter Base Access (5 min)

9. Shifter base is now exposed: metal bracket, stalk, two cables running forward to the trans, rubber dust boots.
10. **Take a photo.** Cable routing and clip orientation matter.
11. Peel back the cable boots at the shifter-side ends.

### Phase C: Disconnect Cables from OEM Shifter (10 min)

12. Each cable terminates in a ball socket that clips onto a shifter-base pin via a small C-clip / circlip.
13. Remove retaining clips with needle-nose or circlip pliers (style depends on year — check Acuity's PDF).
14. Pop each cable off its pin; let them hang — they're captured at the firewall.
15. Clips into a labeled bag immediately.

### Phase D: Swap the Shifter Base (15-20 min)

16. Remove the 4 base bolts (12mm or 14mm depending on production run). Factory Loctite, so they'll be stubborn — breaker bar to crack, ratchet to spin.
17. Lift OEM shifter out. Heavier than expected.
18. Scrape gasket residue / dirt from floor-pan mating surface, wipe clean.
19. Drop in the Acuity shifter. Same 4-bolt pattern.
20. Start all 4 bolts hand-tight before torquing any — lets the bracket self-align.
21. Torque to **17 ft-lb** in a cross pattern ([docs/torque-specs.md](../docs/torque-specs.md) "Shift base bolts"). Do NOT overtighten; the floor pan is thin steel and will dimple.

### Phase E: Throw-Position Selection (5 min)

22. The 3-Way has three pivot positions:
    - **Position 1 (longest throw, closest to OEM):** ~20% reduction
    - **Position 2 (middle):** ~30-35% reduction
    - **Position 3 (shortest):** ~40% reduction
23. **Start at Position 2.** See "Tips on throw-position choice" below. Do NOT jump to Position 3 day one.
24. Remove the C-clip, slide the pivot pin, reposition the lever, re-insert pin, re-install clip.
25. Verify the clip is fully seated. If it pops off while driving, the shifter comes apart in my hand.

### Phase F: Reconnect Cables (10 min)

26. Pop each cable end onto its Acuity pin.
27. Install retaining clips (new from Acuity if supplied).
28. **Tug-test each cable end.** A cable that pops off while driving = no shifting. Catastrophic.
29. Re-seat cable boots.

### Phase G: Trans-Side Bushing Replacement (30-45 min)

30. Lift front, 2 jack stands on subframe.
31. Locate the shift cable bracket on the upper-front face of the trans (behind the intake manifold area). Two cables run to a selector lever on top of the trans and a second selector behind it.
32. Each cable end has:
    - An outer-cable retainer clip (housing to bracket)
    - An inner-cable ball clip (ball-end into the trans selector)
    - An OEM rubber bushing between ball and selector — **this is the replacement target**
33. Work one cable at a time. Don't disconnect both or orientation gets scrambled.
34. Remove the outer clip first (needle-nose). Pocket it — do NOT set it on the subframe.
35. Pop the inner ball clip. The cable end lifts off.
36. Pry out the old OEM bushing. On a 170k car it'll be deformed or torn. Wipe the selector hole clean.
37. **Thin film of silicone grease** on the new Acuity bushing — too much attracts grit.
38. Press the new bushing into the selector hole. Firm thumb pressure. If I'm hammering, stop and check orientation.
39. Seat the cable ball back into the bushing.
40. Inner clip, then outer clip. Tug-test.
41. **Repeat for the second cable.**

These are trans-side bushings only — the bracket isolators stay OEM. Acuity's shifter-side bushing kit is separate and I didn't buy it.

### Phase H: Reinstall (15-20 min)

42. Double-check no tools or clips in the shifter tunnel.
43. Reverse Phase A: console in, 12V outlet reconnected, console screws hand-tight + snug (NOT torqued — plastic threads).
44. Bezel / boot back on — align tabs, press firmly all the way around.
45. Shift knob hand tight.
46. Reconnect battery (-) if disconnected.

### Phase I: Calibration Test — engine OFF, car stationary (5 min)

47. Key off, work the shifter through every gear slowly. Feel for binding, graunch, or a gate that won't find home.
48. **Every gear finds its gate cleanly, including reverse.** Reverse lockout (press-down) must still work.
49. If a gear feels blocked or the lever sits wrong at neutral, STOP. Cable end not seated, or cables crossed at the trans selector.
50. Engine running, clutch in: step 1→6 and back. Smooth, no fight.

## First Drive Checklist

Before leaving the driveway:

- [ ] All 6 gears + reverse engage without force
- [ ] Reverse lockout still requires pressing down
- [ ] Shifter self-centers in neutral — no drift
- [ ] No new rattles / buzzes from the console at idle
- [ ] No tugging / resistance as the engine rocks on its mounts at idle (would indicate a cable routed too tight)

First 5 miles:

- [ ] 1→2 shift clean (this is the most common "notchy after short-shifter" surface)
- [ ] 3→2 downshift clean (reveals bushing pre-load issues)
- [ ] No cable rub — squeaky / rubbery noise on shift = boot or cable hitting bodywork or heat shield
- [ ] Park, get under the car, confirm trans-side bracket secure and no clips walked loose

After 50 miles:

- [ ] Re-torque shifter base bolts to 17 ft-lb. Standard practice for anything heat-cycled under load.
- [ ] Re-inspect trans-side clips. Any clip off its groove = replace.

## Common Mistakes

### Cable pre-load on the trans side

No torque spec on the bushings; they press in. But the outer-cable housing clip can get clocked "one slot over" from where OEM sat, leaving the cable under permanent tension. Symptoms:

- Shifter resting off-center at rest
- One gear easy, the other end of the same gate needs effort (e.g., 1st easy, 2nd fights)
- Accelerated bushing wear

Fix: reposition the outer-cable clip so the cable is relaxed with the shifter in neutral.

### Over-torquing the shifter base bolts

17 ft-lb. Not 30, not "snug with my elbow." The floor pan is thin; dimple it and the bracket no longer sits flat, which throws off pivot geometry.

### Picking the shortest throw on day one

Position 3 is genuinely short. Too short for a 170k-mile trans on most drivers. More below.

### Losing the cable clips

Guaranteed if I'm not careful. 4-6 spare OEM clips on hand. Magnetic parts tray under the work area.

### Grease errors

Dry install = squeak within 100 miles. Over-greased = grease collects grit, which abrades the bushing. Thin film, silicone only.

### Skipping the photo before disconnecting cables

Crossed cables or a cable pinched against a heat shield edge will chafe through in months. Photo before, compare after.

## Tips on Throw-Position Choice

This is where reliability-first actually matters — wrong throw can accelerate synchro wear.

### Position 1 — Least aggressive (~20%)

- Still noticeably shorter than OEM but retains mechanical sympathy.
- Best pick if the trans is high mileage (mine is) and synchros feel tired in 2nd or 3rd.
- Lowest mis-shift risk, lowest chance of a missed 3→2 under brakes.
- **The reliability pick if I'm ever unsure.**

### Position 2 — Middle (~30-35%)

- Sweet spot for a street car that occasionally sees canyon / track.
- Where I've been running it.
- More effort per shift than Position 1 but the feel improvement is big.
- Low mis-shift risk IF technique is clean (fully declutched, deliberate gate motion).

### Position 3 — Max (~40%)

- Genuinely aggressive lever ratio. Any sloppy technique shows up as grinding or a refused gate.
- Honda's synchros are soft brass. Too-short-throw + notchy gearbox = accelerated synchro wear at minimum, bent synchro keys if I rush a 1→2 cold.
- Tempting for bragging rights. **Not worth it** on a 170k trans with no rebuild planned.
- Reserved for post-transmission-rebuild with upgraded synchros.

### Rule I'm giving myself

Start at Position 2. If after 500 miles I'm missing gates or crunching, drop to Position 1. Do NOT escalate to Position 3 unless the trans has been freshened.

Shorter throw also means more lever effort (less mechanical advantage). If shift effort is climbing enough that I'm muscling the lever at speed, drop back a position — that's leverage, not a trans problem.
