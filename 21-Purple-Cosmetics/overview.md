## Status

- **Not yet purchased.** Aesthetic layer — add in phases alongside mechanical work.
- **Theme anchor:** Acuity purple oil cap + Acuity ESCO-T6 purple shift knob (complements the Acuity 3-way adjustable shifter already owned).

## Scope and Rule

Engine bay goes purple. Every part here is **optional** — nothing affects function. The reliability-first rule still applies: if a purple variant of a part doesn't exist from a reputable brand, I either powder-coat an OEM/existing part or skip it. I'm not buying no-name Amazon "universal purple" caps that might fail seals or adhesive.

**Reputable brand list for this project:** Acuity, K-Tuned, Hybrid Racing, Mishimoto, Samco Sport, Dress Up Bolts, Skunk2, Password:JDM, NonStopTuning (NST), Koyorad, BC Racing, Radium Engineering.

## Custom Work Required (no purple option exists)

These need powder-coating or paint because no reputable brand sells them in purple:

- **Washer fluid cap** — OEM 38513-SC4-xxx family. No aftermarket purple billet exists for FG2.
- **Power steering reservoir cap** — OEM 53697-SB3-952. No aftermarket purple billet.
- **Brake master cylinder cap** — OEM 46662-S9A-003. Verus/MDR make billet BMC caps but only for newer generations (FK8+, FL5). Werks JDM has an FG2 reservoir tank upgrade ($180) but it's the whole canister, not a cap.
- **Stock K20Z3 valve cover** — no authentic purple aftermarket cover exists. Spoon is yellow only. Authentic Mugen is red wrinkle only. eBay "purple Mugen" covers are fakes. Skunk2 magnesium cover has cowl-clearance issues on FG2 per Skunk2's own fitment notes.
- **Megan Racing front strut bar** — already owned in polished finish. Powder-coat before install.

**Batching strategy:** drop the washer cap + PS cap + BMC cap + strut bar + valve cover off at ONE local powder coater in one trip. Combined cost ~$330-565. Request candy/illusion purple with clear top coat.

## Parts with NO Purple Option (don't buy cheap substitutes)

- **BC Racing BR coilover springs** — only black from BC. Aftermarket 2.5"/65mm springs in purple don't exist from Swift, Eibach, H&R, Tein, or Hyperco (Hyperco will do custom run but $400+). Accept black.
- **Radiator cap (Koyorad)** — no purple factory option. Powder-coat SK-C13 to purple.
- **Silicone hose clamps (T-bolt)** — stainless only from Vibrant, HPS, Mishimoto, Hybrid Racing. Hymee-brand purple T-bolts on Amazon have corrosion issues — don't trust them on engine coolant. Keep stainless.
- **Header / exhaust accents** — paint fails at exhaust temp. Stay thermally-correct (ceramic coat silver, not purple).
- **Caliper covers** — MGP doesn't stock purple as a standard color. Custom PPG paint match available but $400+ (see `04-Brakes/purple-calipers.md`). G2 paint kit is the recommended approach instead.

## Install Strategy

Purple cosmetics are the **last layer** of each mechanical phase:

1. **Phase 09 (Cooling):** order Samco purple hose kit day-1 (4-6 week lead); install with cooling refresh.
2. **Phase 15 (Pulleys):** order NST alt+idler with ATI Super Damper; install together at pulley phase.
3. **Engine bay dress-up (anytime after headers installed):**
   - Install Acuity oil cap (5 min)
   - Install K-Tuned dipstick (5 min, includes spring retention)
   - Install Dress Up Bolts + Skunk2 VC hardware (2-3 hrs — requires removing intake manifold for VC bolt access if doing the full kit)
   - Install shift knob (2 min)
4. **Batch powder-coat weekend:** drop valve cover + strut bar + fluid caps in one trip.
5. **Fender washers + accents:** install whenever.

## Things to Avoid

1. **Amazon "universal purple" caps.** The threads are close but not correct on FG2 necks — they leak, or worse, pop off under pressure.
2. **eBay "purple Mugen" valve covers.** Fakes. Real Mugen K-series covers are red wrinkle only.
3. **Skunk2 magnesium valve cover.** Not even in purple — Skunk2 themselves flag cowl interference on 06-15 Civic.
4. **Plasti-Dip on anything that sees heat** (calipers, valve cover, exhaust). Sub-30-day failures are typical.
5. **Mixing Engine Ice coolant with Honda Type 2.** Chemistry is incompatible. See `09-Cooling-System/overview.md`.
6. **Cheap purple-anodized T-bolt clamps.** Corrode under coolant exposure. Stainless is the right call.
