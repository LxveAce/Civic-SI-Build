# Torque Specs — K20Z3 / FG2 Quick Reference

> I got tired of hunting forums at 11pm for torque values. This is my cached reference for every fastener I'm likely to touch during this build. Specs pulled from the Honda factory service manual (FSM) and cross-checked with 8thcivic.com community threads.

**Quick sanity check rule:** if I'm not sure and can't find the spec, don't guess on anything load-bearing. OEM specs only. Aftermarket hardware follows aftermarket hardware spec (e.g., ARP), not the part it replaces.

## Fastener Replacement Rules (One-Time Use)

These bolts I replace every disassembly, no exceptions:

- Head bolts
- Flywheel bolts (if Exedy instructions say so)
- Strut-to-knuckle bolts (stretch bolts)
- Lower ball joint pinch bolt
- Axle nuts
- All cotter pins
- All crush washers (banjo bolts, drain plugs)
- All exhaust donut gaskets
- Valve cover gasket (if removed)

## Thread Locker Reference

| Strength | Color | Use |
|----------|-------|-----|
| Low (222) | Purple | Small fasteners, easy removal |
| Medium (242) | Blue | General — flywheel, pulley bolts, suspension | 
| High (262/271) | Red | Permanent — cam gear, VTC actuator. Heat required to remove. |
| Anti-seize | Copper/silver | Exhaust threads, spark plug threads, rotor screws |

**Default to medium (blue 242) unless the spec calls for something else.** Never use red where future disassembly is needed.

## Tools I Need for This Spec List

- 1/2" drive torque wrench, 30-250 ft-lb range (CDI, Tekton, or Snap-On)
- 3/8" drive torque wrench, 10-80 ft-lb range
- 1/4" drive torque wrench, 20-200 in-lb range (for the valve cover and small bolts)
- Calibrated within the last year (I can send it out for recal every ~2 years)
- A harmonic balancer puller (for the crank pulley work)
- An angle gauge for torque-to-yield head bolts (I'm not opening the head, but it's in the kit if I ever do)
